# Vinci
PDF 文档检索工具。

根据用户设定的关键词对本地 PDF 文件进行检索，并生成分析结果。

> **PDF 文件**
>
> 符合 Adobe 官方标准的 PDF，可在浏览器中直接打开查看
>
> **关键词**
>
> 用户自定义，可同时支持1到100个关键词
>
> **分析结果**
>
> - 列表：包含关键词的完整句子（可自定义截取范围）
> - 图表：词频图表
> - 可导出为 Excel (.xls)

## 工具特性
- 单机应用（支持 Windows 、OS X)
- UI 友好
- 高效检索
- 用户使用数据收集（只收集出现异常操作或结果时的数据，用于 Bug 修复和工具升级)
